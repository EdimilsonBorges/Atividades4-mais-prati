# Meu Perfil Pessoal em HTML5 e CSS3

Este projeto foi desenvolvido como parte de um exercício prático das empresas +PRATI junto com a Codifica para consolidar conhecimentos em **HTML5 semântico** e **CSS3 moderno**, com foco em estruturação, estilização e responsividade.

## Estrutura do Projeto

O repositório contém os seguintes arquivos principais na raiz:

- `perfil.html`: Página principal com HTML5 semântico.
- `styles.css`: Estilização utilizando CSS moderno com Flexbox, Grid e responsividade.

## O que foi feito

### HTML5 Semântico

A estrutura do `perfil.html` ( foi substituido por `index.html` para deploy ) segue boas práticas de marcação semântica, utilizando:

- `<header>` com meu nome, foto e slogan.
- `<nav>` com links internos para facilitar a navegação.
- `<main>` como contêiner principal do conteúdo.
- `<section>` separando os blocos: **Sobre mim**, **Habilidades**, **Projetos** e **Contato**.
- `<footer>` com os direitos autorais e créditos.

### CSS

O arquivo `styles.css` inclui:

- **Seletores por elemento, classe e ID**.
- **Flexbox** e **CSS Grid** para criar layouts responsivos e bem organizados.
- **Reset CSS** para uniformizar a aparência entre navegadores.
- **Variáveis CSS** para facilitar manutenção e reutilização de cores e espaçamentos.
- **Pseudo-classes** como `:hover` e `:focus` para interações.
- **Pseudo-elementos** como `::before` e `::after` para efeitos visuais adicionais.
- **Media queries** para garantir uma boa experiência em dispositivos móveis, tablets e desktops.

### Recursos Adicionais

- Favicon personalizado.
- Fontes importadas do **Google Fonts**.
- Efeitos de **transição suave** para uma navegação mais agradável.
